# 前后端分离项目模版
[English](./README.md) | 中文

包含基本的登录、注册、密码重制等等功能，可以二次开发编写具体场景下的应用程序

- 登录功能（支持用户名、邮箱登录）
- 注册用户（通过邮箱登录）
- 重置密码（通过邮箱重置密码）

## 登录功能：

1. 用户登录成功之后，才能访问其他页面
2. 用户如果没有登录，那么会自动跳转到登录页面
3. 如果用户请求的是一个不存在的页面，强制跳转到登录页面，如果已经登录，跳转到首页

解决方案：

1. 无论是否已经登录，直接向后端发送请求
2. 如果请求成功，那么说明已经登录
3. 如果请求失败，那么说明未登录，跳转到登录页面
